如何用hexo和GitHub建立一个个人博客

简单博客需要一个域名和一个放网页文件的空间

1、博客空间。

注册GitHub，并创建repository，命名规则为“username.github.io”。创建的同时初始化readme.md. 此时http://username.github.io/ 应该已经可以访问。如果不可以，在repository首页点击setting，下拉到GitHub pages检查是什么问题。这就是博客空间，免费300M。
在任意终端（Windows是打开cmd，Mac是打开终端）输入 ping username.github.io ，这步是为了得到博客空间的IP地址，用于绑定域名。

2、注册域名。

可以在freenom.com注册免费一年的域名，后缀为.cf .ga .gq .ml 等是免费的。注册好后点击services>my domains. 点击域名，点击freenom dns，添加A记录为ip地址，CNAME记录为username.github.io. 十分钟左右访问域名就能访问到GitHub Page，内容应该为username.github.io，其他啥都没有。

3、网页文件和博客内容。

在本机搭建hexo环境（因为我选择hexo，也可以是其他的）。这里的原理是在本机生成网页文件，然后用git将网页文件推送到GitHub repository，也就是你的博客空间。环境搭建参考hexo官方文档https://hexo.io/zh-cn/docs/

4、文件推送。

连接上GitHub，打开_config.yml（hexo配置文件，以空格为缩进，冒号后面必须有空格）文件最后添加或改写为
deploy:
type: git
repo: https://github.com/username/username.github.io.git
branch: master

推送命令为
hexo clean;(清除缓存)
hexo g;(创建网页文件)
hexo d;（推送到GitHub）
